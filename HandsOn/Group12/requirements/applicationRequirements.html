Application Requirements

General Description

Full Sensor location app for Rivas Vaciamadrid technicians. With it, they will be able to locate the sensors of a certain type,
oriented to carry out maintenance.

Requirements

  • The user can find the sensors easily
  • The user can find the sensors type
  • The user can find where the sensors are exactly
  • The user can find the status of the sensors

Layout

Simple layout of the main page of the app, to serve as an example of the future working system.
When clicking on the search button, the app will display a list of the locations of the sensors of the specified type.
